Want your site to look better when shared in social networks? then this is for you!

Here you can create the necesary Open Graph and Twitter Card code (meta tags) to include in your site.

What are meta tags and why I need them?
This is the bored Wikipedia definition, but in simple words, meta tags are a piece of code that defines standard information about your web page, so other sites like Google, Facebook or Twitter, can display it nicely. They also are helpul for search engine optimizacion aka SEO.

 

Ok I need this in my life, how I use them?
Once you have your meta tags generated, you should include them in the source code of your page, between <head> and </head> tags, and yes, you need to have at least a basic HTML knowledge. HTML is fun, if you don't believe it, ask any webmaster ;)
